# GoogleMapsAPI
Will be update in few weeks include New Design, SearchView, RatingBar, GeoCoding, GeoLocation, and Directions API.
>Its an old project done during my post graduation diploma.
## Tools And Libraries Used:

1. Maps SDK for Android
2. Places API
3. Custom Maps Styling( Night, Retro, Silver, and standard mode)
4. ListView
5. HttpURLConnection
6. AsyncTask 
7. Reading JSON data
 and more.
 
 * ## Application Functions
 On start up application asks for the permission(for getting the location), then it will locate the user
 location on the map.
